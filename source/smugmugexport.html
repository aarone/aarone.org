--- title: SmugMugExport --- SmugMugExport
Upload photos to SmugMug directly from iPhoto!

Features

  • Fast, reliable uploads to SmugMug
  • iPhoto captions and keywords are sent to SmugMug
  • Create, edit, and delete albums
  • Keychain support
  • Universal Binary
  • Optional image scaling during upload to reduce upload times
  • RAW uploads to SmugVault
  • It's free

FAQ

  1. How do I install SmugMugExport?
    1. Download the plugin using the link above
    2. Open your Downloads folder
    3. Double-click the downloaded .zip file
    4. Double-click SmugMugExport.pkg
    5. Follow the steps to install the plugin
    6. Open iPhoto or restart it if it's already open
    7. Select photos in iPhoto
    8. Select the menu 'File', 'Export..'
    9. select the SmugMug tab in the export window and export photos
  2. I'm using Mac OS X 10.8 (Mountain Lion) and I can't use or install the plug-in; does the plugin work?
    Mac OS 10.8 has security restrictions which prevent applications which are not signed by Apple to run by default. If you'd like to run the plugin, you can disable the security restrictions as follows:
    1. Right-click the downloaded .pkg (see installation instructions above)
    2. select 'Open'
    3. A dialog will appear stating that the application is from an unidentified developer
    4. Click 'Open'
    5. Complete the installation
  3. How much does SmugMugExport cost?
    It's free!
  4. iPhoto crashed during exporting, how should I report the bug?
    Email me a copy of the file ~/Library/Logs/CrashReporter/iPhoto.crash.log ('~' is your home)
  5. How do I uninstall SmugMugExport?
    To uninstall, do the following:
    1. Quit iPhoto
    2. select iPhoto in the Finder
    3. Control-click on iPhoto and select 'Show Package Contents'
    4. In the window that opens, navigate to 'Contents/PlugIns'
    5. move SmugMugExport.iPhotoExporter to the trash
  6. Why aren't iPhoto image titles being uploaded?
    Smugmug has a notion of filenames but not titles for images. SmugMugExport currently uses an image's filename as the Smugmug filename attribute during upload. Version 1.0.4 of SmugMugExport allows the iPhoto title to be used as the SmugMug filename.
  7. In which language is SmugMugExport written?
    Objective-C
  8. Which version of SmugMugExport am I running?
    The preferences window displays the current version in the lower-left corner.
